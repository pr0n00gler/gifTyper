# GIFTyper

Converts text to animated typing gifs

### Demo 
![Demo](out.gif)

### Dependencies
* Go 1.13+

### Installation
```
make install
```

### Usage
```shell
gifTyper -text="Hello World!" -output="out.gif"
```
